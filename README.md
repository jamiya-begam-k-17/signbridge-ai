# SignBridge AI

An AI-powered platform for two-way sign language communication and accessible learning, built with React, FastAPI, and computer vision.

## Features

- Real-time sign language recognition using computer vision
- Two-way communication between sign language and text/speech
- User authentication and conversation history
- Classroom mode for educational purposes
- Responsive web interface

## Installation

### Backend

1. Navigate to the backend directory:
   ```
   cd backend
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Run the server:
   ```
   python main.py
   ```

### Frontend

1. Navigate to the frontend directory:
   ```
   cd frontend
   ```

2. Install dependencies:
   ```
   npm install
   ```

3. Start the development server:
   ```
   npm run dev
   ```

### AI Model

1. Navigate to the ai_model directory:
   ```
   cd ai_model
   ```

2. Follow the training scripts for model setup.

## Usage

- Access the web application through the frontend.
- Use the camera for sign language detection.
- Switch between translation and classroom modes.
